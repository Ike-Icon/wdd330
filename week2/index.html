Programming Basics

Commenting Codes: It's a good habit to comment on your code for others to read and understand. Using // to comment a line and /* for more than 1 line.

A Block of code is a series of statements that are collected together inside curly braces.

Primitive Data Types: There are six data types that are primitive and are:

  • String
  • Symbol
  • Number
  • Boolean
  • Undefined
  • Null

Template Literals: They allow interpolation, this means that JS expression can be inserted insie a string, eg `Hello ${name}!`;

Arrays, Logic, and Loops

Arrays are an ordered list of values, e.g., const myArray = ['one', 2, 'five']

Destructuring Arrays: It is the concept of taking values out of an array and presenting them as individual values. e.g., const [x, y] = [1, 2]; where x = 1 and y = 2.

    Array Methods
  • To find the array length >> myArray.length;
  • To remove the last item >> myArray.pop();
  • To remove the first item >> myArray.shift();
  • To add a new item to the end of the array >> myArray.push('item');
  • To add a new item to the beginning of the array >> myArray.unshift('item');

Arrays can be merged with the use of the "concat()" method and with the use of the "join()" method, arrays can be turned into strings.

Functions

A function is a chunk of code that can be referenced by a name, and is almost like a small, self-contained mini program. Functions can help reduce repetition and make code easier to follow.

Parameters and Argumentss are terms that are often used interchangeably to represent values provided for the function as an input. eg.function square(x){ return x*x; }
